"Object-Oriented Watches System" — симулятор умных часов

### 1. Запуск программы
1. Запустите исполняемый файл программы.
2. Появится заставка.
3. Нажмите **ENTER** для продолжения.
4. Вы попадёте в главное меню.

### 2. Главное меню
1. Новый сценарий (создать пользователя + запустить сценарий)
2. Об авторе
3. Руководство пользователя (это руководство)
4. О программе
5. Настройки
6. Загрузить сессию
7. Выход из программы

### 3. Создание пользователя
- Введите имя пользователя
- Введите GTM (смещение часового пояса)
- Введите географическое положение (город/регион)

### 4. Доступные сценарии

**1. Полёт (Смена часовых поясов)**
- Укажите время отправления
- Укажите продолжительность полёта
- Укажите пункт назначения и его GTM
- В процессе полёта можно:
  - Симулировать время (30 или 60 минут)
  - Сменить часовой пояс
  - Прибыть в пункт назначения

**2. Простое погружение (DivingWatches)**
- Укажите время начала погружения
- Укажите максимальную глубину
- Установите безель
- Выберите тип часов
- После начала можно наблюдать за изменением времени

**3. Dive Computer (Профессиональное погружение)**
- Укажите время, максимальную глубину и температуру воды
- После спуска доступны действия:
  - Оставаться на глубине (15 мин)
  - Всплыть на 1 метр
  - Всплыть на 5 метров
  - Аварийное всплытие
  - Завершить погружение

### 5. Сохранение и загрузка
- После каждого сценария сессия автоматически сохраняется.
- В главном меню можно загрузить последнюю сессию (пункт 6).

### 6. Настройки
- Смена языка (ru / en)
- Изменение пути к файлам (в разработке)

### Полезные советы:
- Время в программе отображается в формате ЧЧ:ММ
- При всплытии и пребывании на глубине время автоматически увеличивается
- Все действия логируются в файл logs.txt
- Не забудьте сохранять сессию перед выходом
